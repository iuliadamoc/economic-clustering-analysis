# 📊 Economic Clustering Analysis

## 📖 Overview
This project performs clustering analysis on socio-economic indicators to identify patterns and similarities between countries.

The goal is to group countries based on economic and social metrics and uncover hidden structures in the data.

## 🚀 Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Dataset
The dataset includes indicators such as:
- GDP per capita
- Life expectancy
- Inflation
- Income
- Health expenditure

## 🔍 Methodology
1. Data cleaning & preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature scaling  
4. Clustering using K-Means  
5. Evaluation using silhouette score  

## 📈 Results
- Identified X clusters of countries
- Cluster 1: High income, low mortality
- Cluster 2: Developing economies
- Cluster 3: Underdeveloped regions
